## What Is an Embedding?

An embedding is a mapping from discrete symbols (words, tokens, entities) into continuous vector space. It captures semantic relationships in a format that neural networks can process.

Embeddings are the foundation of modern NLP systems and enable models to reason about meaning and similarity.
