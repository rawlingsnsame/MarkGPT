## What Is a Vector Space?

A vector space is a mathematical structure formed by vectors that can be added together and scaled. Embeddings live in vector spaces where semantic relationships become geometric relationships.

Understanding vector spaces helps you reason about similarity, analogy, and the behavior of embedding-based models.
