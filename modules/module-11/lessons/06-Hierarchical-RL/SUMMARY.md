# Lesson 6: Summary

Hierarchical RL decomposes complex problems into hierarchies of subtasks. Options provide temporal abstraction enabling multi-timescale decision-making. Credit assignment through hierarchies remains challenging but enables skill reuse and better generalization. Well-designed hierarchies exponentially reduce problem complexity.

**Core Learning Outcomes:**
- Temporal abstraction through options enables hierarchical planning
- Skill reuse accelerates learning on similar problems
- Credit assignment through hierarchy levels is challenging but solvable
- Option discovery requires domain knowledge or auxiliary signals
- Multi-level hierarchies enable solutions to very complex tasks

For practitioners: If natural task decomposition exists (high-level planner + low-level controller), use hierarchy explicitly.
