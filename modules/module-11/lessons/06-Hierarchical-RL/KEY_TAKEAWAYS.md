# Lesson 6: Key Takeaways and Summary

## Hierarchical RL Foundations

1. **Temporal abstraction**: Options extend over multiple timesteps.
2. **Skill reuse**: Learned subtasks transfer across problems.
3. **Options framework**: Initiation set, policy, termination condition.
4. **Credit assignment**: Distributing credit through hierarchy levels.
5. **Option discovery**: Automatic learning of useful skills.
6. **Feudal learning**: Manager-worker hierarchies with goal-setting.
7. **Multi-level hierarchies**: Complex problems decompose into many levels.
8. **HAMs**: Explicit hierarchical state machines encoding task structure.

## Key Algorithms

- **HIRO**: Learn low-level  policy achieving manager-set goals.
- **MEGA**: Mutual-information maximization for skill discovery.
- **SkillTree**: Hierarchical skill learning and reuse.

## Benefits and Challenges

**Benefits**: Exponential reduction in decision complexity, skill reuse, interpretability.
**Challenges**: Credit assignment through levels, determining right hierarchy, discovering options automatically.
