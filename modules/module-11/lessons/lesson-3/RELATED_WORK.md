# Related Work on Multi-Agent Reinforcement Learning

## Foundational MARL
- Littman, M. L. (1994). Markov games as a framework for multi-agent reinforcement learning
  - Game-theoretic foundations
  - Nash equilibrium learning

## Communication and Coordination
- Foerster et al. (2016). Learning to communicate with deep multi-agent reinforcement learning
  - Communication protocol learning
  - Discrete communication channels

## Value Decomposition
- Rashid et al. (2018). QMIX: monotonic value function factorization
  - Value decomposition for cooperative MARL
  - Theoretical guarantees

## Centralized Training, Decentralized Execution
- Foerster et al. (2018). Counterfactual multi-agent policy gradients
  - COMA: credit assignment via counterfactuals
  - Addresses free-rider problem

## Independent Learners
- Claus, C., & Boutilier, C. (1998). The dynamics of reinforcement learning in cooperative multiagent systems
  - Independent learners
  - Convergence conditions

## Applications
- Jaderberg et al. (2018). Human-level performance in 3D multiplayer games
  - Competitive multi-agent learning
  - Self-play training
