# Lesson 5: Key Takeaways and Summary

## Meta-Learning Essentials

1. **Learning to learn**: Improve learning algorithm across task distribution.
2. **MAML**: Learn good initializations for rapid task adaptation (bi-level optimization).
3. **Few-shot learning**: Solve new tasks from minimal data.
4. **Task distribution**: Diversity of meta-training tasks determines generalization.
5. **Metric learning**: Learn similarity functions for rapid adaptation.
6. **Prototypical networks**: Class prototypes as central representations.
7. **Matching networks**: Attention-based learned matching processes.
8. **Domain generalization**: Handle distribution shift through diverse task training.

## algorithmic approaches

- **Optimization-based**: Learn update rules or initializations (MAML, FOMAML).
- **Metric-based**: Learn distance functions (Prototypicals, Matching Networks).
- **Recurrent/generative**: Meta-networks generating task-specific parameters.

## When meta-learning applies

- Few-shot classification and NLP tasks.
- RL: Rapid policy adaptation to new environments/tasks.
- Robotics: One-shot learning of manipulation skills.
- Effective when task diversity exists during training.
