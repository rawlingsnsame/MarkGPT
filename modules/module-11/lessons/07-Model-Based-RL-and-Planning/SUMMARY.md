# Lesson 7: Summary

Model-based RL explicitly learns environment models and uses them for planning. Dyna combines real and imagined experience for improved sample efficiency. MPC enables online optimization of action sequences using learned models. Latent space models enable efficient planning in high-dimensional spaces. Model uncertainty remains a key challenge for long-horizon planning.

**Core Learning Outcomes:**
- World models enable learning-efficient planning
- Dyna combines model-free and model-based learning synergistically
- Imagination enables learning from simulated experience
- Latent models work better than pixel models for planning
- Model uncertainty quantification enables robust planning

For practitioners: Model-based methods excel when sample efficiency is critical; start with simple ensemble models.
