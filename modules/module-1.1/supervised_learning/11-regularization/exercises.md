# Regularization - Exercises

## Exercise 1: L1 vs L2 Regularization
Train linear regression models with L1 (Lasso), L2 (Ridge), and no regularization. Compare coefficients and model performance.

## Exercise 2: Regularization Parameter Tuning
Use cross-validation to find optimal alpha parameter for Ridge and Lasso. Plot alpha vs. CV score.

## Exercise 3: Elastic Net Regularization
Implement Elastic Net combining L1 and L2. Experiment with different L1 ratios and compare with pure L1 and L2.

## Exercise 4: Dropout in Neural Networks
Implement dropout layers in a neural network. Compare models with and without dropout to demonstrate overfitting prevention.

## Exercise 5: Early Stopping Implementation
Train a neural network with early stopping. Monitor validation loss and stop training when it plateaus.

## Exercise 6: Learning Curves for Regularization
Plot learning curves showing how regularization affects bias-variance trade-off for a linear model.

## Exercise 7: Feature Importance with L1
Train a Lasso model and identify which features have zero coefficients. Discuss feature selection properties.

## Exercise 8: Regularization in Tree Models
Train decision trees with and without regularization (max_depth, min_samples_split). Show overfitting prevention.

## Exercise 9: Cross-Validation with Regularization
Use k-fold cross-validation to select optimal regularization parameters. Compare with single train-test split.

## Exercise 10: Real-World Regularization Strategy
Build a complete model with multiple regularization techniques applied. Select optimal combination for production use.
