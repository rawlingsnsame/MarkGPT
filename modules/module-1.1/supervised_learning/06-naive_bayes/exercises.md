# Naive Bayes - Exercises

## Exercise 1: Basic Gaussian Naive Bayes
Train a Gaussian Naive Bayes classifier on the Iris dataset. Evaluate with accuracy, precision, recall, and F1-score.

## Exercise 2: Multinomial Naive Bayes for Text
Load a text classification dataset (e.g., 20 newsgroups). Build a TF-IDF vectorizer and train Multinomial Naive Bayes. Evaluate performance.

## Exercise 3: Bernoulli Naive Bayes
Train Bernoulli Naive Bayes on binary feature data (e.g., document presence/absence). Compare with Multinomial NB and Gaussian NB on the same data.

## Exercise 4: Feature Independence Assumption
Create a dataset where features are clearly dependent. Train Naive Bayes and a model aware of dependencies. Compare performance and discuss the naive assumption impact.

## Exercise 5: Class Probability Estimation
Train Naive Bayes and extract class prior probabilities and feature likelihoods. Manually calculate predictions for sample data and verify with model predictions.

## Exercise 6: Smoothing Techniques
Train Naive Bayes models with different smoothing parameters (additive/Laplace smoothing). Show how smoothing affects predictions on unseen feature values.

## Exercise 7: Handling Missing Values
Create a dataset with missing values. Implement strategies to handle them in Naive Bayes and evaluate impact on performance.

## Exercise 8: Real-World Text Classification
Implement a spam classification system using Multinomial Naive Bayes. Include data preprocessing, feature extraction, model training, and evaluation with ROC curve.

## Exercise 9: Comparing Naive Bayes Variants
Train all three Naive Bayes variants (Gaussian, Multinomial, Bernoulli) on an appropriate dataset. Create a comparison table and discuss suitability.

## Exercise 10: Domain-Specific Application
Apply Naive Bayes to a real-world problem (e.g., sentiment analysis, medical diagnosis). Include data exploration, preprocessing, modeling, and interpretation.
