# Feature Selection - Exercises

## Exercise 1: Univariate Feature Selection
Use SelectKBest with chi²/f_classif to select top k features. Visualize feature scores and train models with/without selection.

## Exercise 2: Recursive Feature Elimination
Implement RFE to recursively remove features. Plot feature importance ranking and model performance vs. number of features.

## Exercise 3: Feature Importance from Tree Models
Extract feature importances from Random Forest and Gradient Boosting. Compare feature rankings between models.

## Exercise 4: Correlation Matrix Analysis
Analyze feature correlations and remove highly correlated features. Show impact on model performance and interpretability.

## Exercise 5: L1-based Feature Selection
Train Lasso and extract non-zero coefficients as selected features. Compare with filter and wrapper methods.

## Exercise 6: Wrapper Method with Cross-Validation
Implement forward and backward selection using cross-validation. Compare with univariate selection.

## Exercise 7: Permutation Importance
Calculate permutation importance for features. Compare with built-in importances and identify discrepancies.

## Exercise 8: Dimensionality Reduction vs Selection
Compare feature selection with PCA and t-SNE. Discuss interpretability trade-offs.

## Exercise 9: Domain Knowledge Integration
Apply  domain knowledge to select features. Compare with statistical selection and discuss hybrid approaches.

## Exercise 10: Real-World High-Dimensional Project
Work with high-dimensional data (text, images). Apply multiple feature selection techniques and evaluate impact on model performance and training time.
