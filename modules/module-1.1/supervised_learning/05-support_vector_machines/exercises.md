# Support Vector Machines - Exercises

## Exercise 1: Linear SVM Classification
Build a linear SVM classifier on a linearly separable binary classification dataset. Visualize the decision boundary and support vectors.

## Exercise 2: Kernel Comparison
Train SVM models with different kernels (linear, RBF, polynomial) on a non-linear dataset. Compare decision boundaries and performance metrics.

## Exercise 3: C Parameter Tuning
Train SVM models with varying C values (0.01, 0.1, 1, 10, 100). Show how C affects the margin size and number of support vectors.

## Exercise 4: Feature Scaling Importance
Train two SVM models on the same data: one with raw features and one with scaled features. Demonstrate the importance of scaling for SVM performance.

## Exercise 5: Multi-class SVM
Use One-vs-Rest and One-vs-One strategies for multi-class SVM on the Iris dataset. Compare performance and computational efficiency.

## Exercise 6: SVM Regression
Build an SVM regressor using epsilon-SVR and compare with linear regression on a continuous prediction task. Explain the difference in approach.

## Exercise 7: Support Vector Analysis
Train an SVM and analyze the support vectors. Visualize their location relative to the decision boundary. Discuss their importance in the model.

## Exercise 8: Cross-Validation and Grid Search
Implement nested cross-validation with grid search to find optimal C and gamma parameters for RBF kernel. Report optimal parameters and CV scores.

## Exercise 9: Handling Class Imbalance
Train SVM on an imbalanced binary classification dataset. Use class weights to adjust the penalty and compare with standard SVM.

## Exercise 10: Large-Scale SVM Project
Apply SVM to a high-dimensional real dataset (e.g., document classification, image classification). Address computational challenges and implement practical optimizations.
