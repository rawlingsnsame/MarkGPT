# Neural Networks - Exercises

## Exercise 1: Simple Feedforward Neural Network
Build a simple 2-layer neural network using TensorFlow/Keras for binary classification on a synthetic dataset. Visualize decision boundaries.

## Exercise 2: Network Architecture Exploration
Create neural networks with varying depths (1, 2, 3, 4) and widths for the same classification task. Plot train/test accuracy and discuss overfitting.

## Exercise 3: Activation Function Comparison
Train neural networks with different activation functions (ReLU, Sigmoid, Tanh) in hidden layers. Compare performance and convergence speed.

## Exercise 4: Learning Rate and Optimization
Train networks with different learning rates and optimizers (SGD, Adam, RMSprop). Analyze convergence curves and final performance.

## Exercise 5: Regularization Techniques
Implement L1, L2 regularization and Dropout in neural networks. Compare their effect on overfitting and generalization.

## Exercise 6: Batch Normalization Impact
Train two neural networks: one with and one without batch normalization. Measure improvement in training speed and convergence.

## Exercise 7: Neural Network Regression
Build a neural network for continuous target prediction. Compare with linear regression and tree-based methods.

## Exercise 8: Hyperparameter Tuning with Keras Tuner
Use Keras Tuner to automatically search for optimal architecture (layers, units, activation) and training parameters.

## Exercise 9: Visualizing Learning
Monitor loss and accuracy during training using callbacks. Plot learning curves and use techniques like saliency maps to understand what the network learns.

## Exercise 10: Real-World Deep Learning Project
Build a complete neural network pipeline on a real multi-class classification problem. Include data preprocessing, augmentation, architecture design, training, evaluation, and prediction analysis.
