# K-Nearest Neighbors - Exercises

## Exercise 1: Basic KNN Classification
Implement KNN classifier on the Iris dataset with k=3, 5, 7. Evaluate accuracy and visualize decision boundaries in 2D.

## Exercise 2: Optimal K Selection
Train KNN models with k values from 1 to 20 on a classification dataset. Plot accuracy vs. k and identify the optimal k using cross-validation.

## Exercise 3: Distance Metrics Comparison
Train KNN with different distance metrics (Euclidean, Manhattan, Minkowski) on the same dataset. Compare performance and explain differences.

## Exercise 4: Feature Scaling Impact
Train two KNN models: one with raw features and one with scaled features. Demonstrate how feature scaling affects distance-based predictions.

## Exercise 5: Weighted vs. Uniform KNN
Train KNN models with uniform and distance-weighted voting. Compare predictions on boundary points and discuss the impact of weighting.

## Exercise 6: KNN Regression
Build a KNN regressor for continuous output prediction. Compare with linear regression and decision tree regressors.

## Exercise 7: Computational Efficiency
Measure training and prediction time for KNN with increasing dataset sizes. Discuss the computational implications and potential optimizations (KD-trees, ball trees).

## Exercise 8: Handling Imbalanced Data
Apply KNN to imbalanced classification data. Experiment with weighted voting based on class distribution and compare results.

## Exercise 9: High-Dimensional Curse
Create datasets of increasing dimensionality and observe how KNN performance degrades. Discuss the "curse of dimensionality" and potential solutions.

## Exercise 10: Real-World Application with Optimization
Apply KNN to a real dataset. Implement optimizations for efficiency (spatial indexing), tune k and distance metrics, and evaluate on a hold-out test set.
